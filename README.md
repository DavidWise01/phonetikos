# PHŌNĒTIKOS · PHN

**φωνητικός** *(phōnētikos, "of the voice")*, from **φωνή** *(phōnē, "voice, sound")* —
a UD0 universe where each **word** is an emergent and each entry is a rigorous,
honestly-sourced **etymological green paper**: cognate sets, sound laws, dated
attestations, and a hard line against folk etymology.

> Every word is a fossil. Here we read the bone, not the bedtime story.

### The codex
- **№1 — FUCK** (`papers/fuck.html`): the full history & lineage of English's most
  versatile and most policed word. Germanic to the bone (Dutch *fokken*, German *ficken*);
  the acronym myths debunked; earliest secure evidence (1310 / the 1475 cipher poem); the
  ~170-year dictionary exile and 1972 OED re-entry; the grammar (incl. *abso-fucking-lutely*);
  Mohr's Holy→Shit taboo shift; and an honest **Real-or-Fluff** on every claim.

Each word carries a full `.dlw` ACI badge. Sources: OED; Sheidlower, *The F-Word*;
Mohr, *Holy Sh\*t*; Watkins; Liberman; Booth (2015); Hughes.
